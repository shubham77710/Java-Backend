# Day 25 - Collection Framework Overview

## Categories in Collection Framework:

### 1. List (Dynamic Array & Linked Structure)
- ArrayList → Dynamic Array
- LinkedList → Doubly Linked List

### 2. Set (No Duplicates)
- HashSet → Unordered
- LinkedHashSet → Maintains insertion order

### 3. Stack (LIFO)
- Stack → Last In First Out

### 4. Queue (FIFO)
- Queue → Interface
- ArrayDeque → Efficient queue implementation

### 5. Map (Key-Value Pair)
- HashMap → Unordered
- LinkedHashMap → Ordered
- TreeMap → Sorted keys

### 6. Tree-based Structures
- TreeSet → Sorted unique elements
- TreeMap → Sorted key-value pairs

## Key Concepts:
- List allows duplicates
- Set does not allow duplicates
- Map stores key-value pairs
- Tree structures maintain sorted order
